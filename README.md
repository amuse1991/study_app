# 스터디타임 어플리케이션

## 프로젝트 개요
+ 시간표 형태로 스터디를 생성 및 참여가 가능한 어플리케이션 개발

## 팀원 및 역할
+ 신윤호 : 백엔드 개발
+ 진승언 : 어플리케이션 개발

## 사용 기술
+ application : android
+ backend : spring boot
+ DB : MySQL, JPA, AWS RDB
+ server : AWS EC2 Ubuntu 16.04

## 스터디 목표
+ 안드로이드 개발
+ spring boot 및 JPA(Hibernate)를 이용한 개발
+ TDD 기반 개발
+ REST API server 구축
+ devOps 환경 구축
+ 앱스토어에 어플리케이션 출시
+ github를 이용한 협업
